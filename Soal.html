  • 1. Apa perbedaan utama antara var, let, dan const dalam JavaScript?
  • Jawaban: var adalah function-scoped, let adalah block-scoped dan dapat di-update, const adalah block-scoped dan tidak dapat di-update.
  • 2. Apa output dari kode berikut?
  •             var x = 10; 
            let y = 5; 
            const z = 20; 
            x = 15; 
            y = 10; 
            z = 25; 
            console.log(x, y, z); // Output: 15, 10, Error

  • 7. Apa hasil dari operasi berikut?
  •             console.log(10 + '5'); // Output: '105'
            console.log(10 - '5'); // Output: 5
            console.log(10 === '10'); // Output: false
            console.log(10 == '10'); // Output: true
            console.log(true && false); // Output: false
            console.log(true || false); // Output: true
            console.log(!true); // Output: false
  • 8. Buat kode yang menggunakan operator perbandingan untuk mengecek apakah sebuah angka lebih besar dari 100 dan lebih kecil dari 500!
  •             let number = 250;
            if (number > 100 && number < 500) {
                console.log('Memenuhi Syarat');
            }
  • 9. Buat program if-else untuk menentukan apakah sebuah angka ganjil atau genap.
  •             let num = 4;
            if (num % 2 === 0) {
                console.log('ganjil');
            } else {
                console.log('genap');
            }
  • 10. Gunakan switch-case untuk mencetak nama hari berdasarkan nomor (1 = Senin, 2 = Selasa, ... 7 = Minggu).
  •             let hari = 3;
            switch (hari) {
                case 1: console.log('senin'); break;
                case 2: console.log('selasa'); break;
                case 3: console.log('rabu'); break;
                case 4: console.log('kamis'); break;
                case 5: console.log('jumat'); break;
                case 6: console.log('sabtu'); break;
                case 7: console.log('minggu'); break;
                default: console.log('Invalid hari');
            }
  • 11. Gunakan for loop untuk mencetak angka dari 1 hingga 10.
  •             for (let i = 1; i <= 10; i++) {
                console.log(i);
            }
  • 12. Gunakan while loop untuk mencetak angka dari 10 hingga 1.
  •             let i = 10;
            while (i >= 1) {
                console.log(i);
                i--;
            }
  • 13. Gunakan do-while loop untuk mencetak angka dari 1 hingga 5.
  •             let j = 1;
            do {
                console.log(j);
                j++;
            } while (j <= 5);
